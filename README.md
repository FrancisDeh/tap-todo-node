### TAP Todo with Node js

### Setup project
```
git clone git@github.com:FrancisDeh/tap-todo-node.git

cd tap-todo-node

copy .example.env .env
```
### Install dependencies
```
npm install
```

### Create database
Go to mongo db online and create db.
Replace the `.env` with the correct db url.

### Run the project
```
npm run start
```
